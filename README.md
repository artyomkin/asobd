### Курсовой проект

Еще не доработан. В финальной версии будет развертывание по 1 кнопке, дашборды в графане, инструкции по запуску и диаграммы

- scripts - скрипты для операций с хранилищем и витринами
- src - исходный код
  - gen_clicks.py - генерация событий
  - server.py - прием http запросов с событиями и поднятие пайплайна
  - etl.py - etl пайплайн для обработки событий
  - consumer.py - скрипт читающий кафку и запускающий пайплайн
